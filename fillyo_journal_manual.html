Firebase 프로젝트 생성
console.firebase.google.com — 소요 시간 약 5분
1
Firebase 콘솔 접속
console.firebase.google.com 접속 → 구글 계정으로 로그인 → 프로젝트 만들기 클릭

2
프로젝트 이름 입력
프로젝트 이름을 영어로 입력 (예: my-dojang) → 계속 클릭 → Google Analytics는 사용 안함으로 체크 해제 → 프로젝트 만들기
💡 프로젝트 이름은 영어 소문자와 하이픈(-)만 사용 가능합니다. 예: taesarang-dojang, my-hakwon
✅ Google Analytics 사용 여부는 사용 안함으로 선택해도 됩니다. 업무일지에는 필요 없어요.

Realtime Database 설정
데이터 저장소 만들기 — 소요 시간 약 3분
1
Realtime Database 메뉴 찾기
프로젝트 생성 완료 후 → 왼쪽 메뉴에서 빌드 → Realtime Database 클릭
console.firebase.google.com/project/내프로젝트
🏠 프로젝트 개요
빌드
🔐 Authentication
📊 Realtime Database ← 클릭!
📦 Firestore Database
💾 Storage
🌐 Hosting
Realtime Database
실시간으로 데이터를 저장하고 동기화하세요
데이터베이스 만들기
👆 왼쪽 메뉴 빌드 섹션 → Realtime Database 클릭
2
데이터베이스 만들기
데이터베이스 만들기 클릭 → 위치는 Singapore (asia-southeast1) 선택 → 보안 규칙은 테스트 모드 선택 → 사용 설정
⚠️ 위치는 반드시 Singapore (asia-southeast1)를 선택하세요. 한국에서 가장 빠르게 연결됩니다.
3
보안 규칙 설정 (중요!)
데이터베이스 생성 후 → 규칙 탭 클릭 → 아래 코드로 교체 → 게시 클릭
// Realtime Database 규칙 — 아래 내용으로 교체 { "rules": { ".read": true, ".write": true } }
✅ 이 설정은 팀원들이 비밀번호 없이 데이터를 읽고 쓸 수 있게 해줍니다. 앱 자체에서 PIN 번호로 보호됩니다.
4
Firebase 설정값 확인 (복사할 정보)
왼쪽 메뉴 상단 ⚙️ 프로젝트 설정 → 일반 탭 → 아래로 스크롤 → 내 앱 섹션 → 웹 앱 추가 ( 아이콘) 클릭
console.firebase.google.com/.../settings/general

5
firebaseConfig 값 복사
앱 닉네임 입력 (예: dojang-app) → 앱 등록 → 아래와 같은 설정값이 나타납니다. 각 값을 복사해두세요.
// 이런 형태로 나타납니다 — 각 값을 복사하세요 const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXX", authDomain: "내프로젝트.firebaseapp.com", databaseURL: "https://내프로젝트-default-rtdb.asia-southeast1.firebasedatabase.app", projectId: "내프로젝트", };
⚠️ 이 값들은 다음 단계에서 업무일지 초기 설정 화면에 입력해야 합니다. 메모장에 복사해두세요!

2
설정값 입력
도장 이름과 앞 단계에서 복사한 Firebase 값 5가지를 입력 → 설정 완료 — 업무일지 시작하기 클릭
  • 도장·학원 이름 입력 (예: 태사랑 태권도)
  • Firebase API Key 붙여넣기
  • Auth Domain 붙여넣기
  • Database URL 붙여넣기
  • Project ID 붙여넣기
✅ 설정 완료 버튼을 누르면 PIN 번호 설정 화면이 나타납니다. 팀원 모두가 사용할 4자리 번호를 설정하세요.

팀원과 공유하기
사범님들 스마트폰에 바로 설치
1
GitHub Pages 주소를 카카오톡으로 공유
설정이 완료된 후 https://내아이디.github.io/dojang-journal 주소를 사범님들에게 카카오톡으로 공유
✅ 팀원들은 설정 화면 없이 바로 PIN 입력 화면이 나타납니다. 관장님이 설정한 PIN 번호를 알려주세요.
2
스마트폰 홈 화면에 추가 (앱처럼 사용)
스마트폰 브라우저에서 주소 접속 → 아이폰: 공유 버튼(□↑) → 홈 화면에 추가 / 갤럭시: 메뉴(⋮) → 홈 화면에 추가
📱 홈 화면에 추가하면 앱처럼 아이콘이 생기고 전체 화면으로 사용할 수 있어요.
