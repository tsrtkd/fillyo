Getting Started

시작하기 3단계

아래 순서대로 따라오시면 5분 안에 우리 도장 전용 시스템이 준비됩니다.

01
📱
회원가입
fillyo.kr → 로그인에서 이메일 주소와 비밀번호를 입력하세요.
별도 앱 설치 없이 브라우저에서 바로 사용합니다.
02
🏫
도장 설정
로그인 후 도장 설정(Setup)에서 도장 이름과 운영 정보를 입력합니다.
부 수(1~10부) 등 우리 도장에 맞게 설정하세요.
03
🎛️
서비스 선택
대시보드에서 사용할 서비스를 선택합니다.
업무일지는 바로 시작할 수 있고, 승급심사·줄넘기 평가는 구독 후 활성화됩니다.

How to Use

서비스별 사용법

각 서비스를 클릭해 주요 기능과 사용 순서를 확인하세요.

매일 수업 후 출석·결석 인원을 부별로 입력하면 자동 집계됩니다.
입력 중에는 저장되지 않으며, 마지막 입력 후 2.5초가 지나면 자동 저장됩니다.

  1. 1 대시보드 → 업무일지 탭으로 이동합니다.
  2. 2 날짜를 확인하고 부별 출석 / 결석 인원을 숫자로 입력합니다.
  3. 3 차량이 있는 경우 🔵탑승 / 🔴하차 / 📷사진 버튼으로 운행 기록을 남깁니다.
  4. 4 하단 특이사항 란에 상담 내용, 공지사항 등을 자유롭게 메모합니다.
  5. 5 TODO를 입력하면 체크 완료 시 다음날 자동으로 정리됩니다.
  6. 6 ⚙️ 설정 탭에서 부 수(1~10부), 차량 대수(1~5대)를 변경할 수 있습니다.
✓ 2.5초 자동저장 ✓ 실시간 팀 공유 (Pro) ✓ 결석자 자동 집계 ✓ 달력 이력 조회 ✓ CSV 내보내기

우리 도장의 심사 방식에 맞게 항목을 직접 선택하고, 채점 후 결과카드를 자동으로 생성합니다.
학부모님께 투명한 심사 결과를 공유할 수 있습니다.

  1. 1 ⚙️ 설정에서 심사에 사용할 항목을 선택합니다.
    품새 / 발차기 / 인성교육 / 익스트림 중 선택
  2. 2 승급 단위를 설정합니다 — 1개 급씩 또는 2개 급씩 올릴 수 있습니다.
  3. 3 심사 대상 학생을 불러오고 항목별 점수를 입력합니다.
  4. 4 채점이 완료되면 결과카드를 생성하고 저장·공유합니다.
  5. 5 학생별 승급 이력이 자동으로 누적 기록됩니다.
✓ 맞춤 심사 항목 ✓ 자동 결과카드 ✓ 승급 이력 누적 ✓ JPG 결과카드 저장

학생의 도전 급수를 입력하면 현재 급수에서 다음 급수가 자동으로 표시됩니다.
우리 도장만의 기준을 직접 등록하거나 기본 제공 급수표를 사용할 수 있습니다.

  1. 1 ⚙️ 설정 탭에서 급수표를 설정합니다.
    엑셀(.xlsx) 또는 이미지 파일로 업로드하거나 기본 급수표를 사용합니다.
  2. 2 평가 화면에서 학생을 선택하고 도전 급수를 입력합니다.
  3. 3 평가 결과에 따라 현재 급수 → 다음 급수가 자동으로 표시됩니다.
  4. 4 합격 후 🏆 급수증 탭에서 학생을 선택해 급수증을 출력할 수 있습니다.
  5. 5 학생별 급수 이력이 누적되어 언제든 조회할 수 있습니다.
✓ 커스텀 급수표 ✓ 자동 급수 판정 ✓ 급수증 발급 ✓ 이력 누적 관리
